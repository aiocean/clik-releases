# Clik Releases

Download releases for [Clik](https://github.com/aiocean/clik) - a screenshot tool for AI feedback.

## Installation (macOS Apple Silicon)

```bash
curl -L https://github.com/aiocean/clik-releases/releases/latest/download/clik_0.1.0_aarch64.dmg -o ~/Downloads/Clik.dmg
open ~/Downloads/Clik.dmg
```

After downloading, drag Clik to Applications.

> **Note:** Since this app is ad-hoc signed, you may need to right-click and select "Open" the first time, or run `xattr -cr /Applications/Clik.app` in Terminal.
