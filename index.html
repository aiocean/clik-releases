Install

Get Clik in 3 steps

Takes less than a minute. No account required.

Requires Apple Silicon (M1, M2, M3, M4)
1

Open Terminal

Press Space, type Terminal, press Enter

Terminal
2

Copy & paste this command

Click the button below, then paste in Terminal with V

Terminal
curl -L https://github.com/aiocean/clik-releases/releases/download/v0.16.21/clik_0.16.21_aarch64.dmg -o ~/Downloads/Clik.dmg && open ~/Downloads/Clik.dmg
3

Drag to Applications

A window will open. Drag Clik to Applications folder.

Clik Applications
That's it! Launch Clik from Applications and start capturing.
