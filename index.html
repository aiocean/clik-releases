Install

Get started in seconds

Download and install Clik on your Mac with a single command.

1
Open Terminal
Press Cmd + Space, type "Terminal", press Enter
2
Run this command
Terminal 
curl -L https://github.com/aiocean/clik-releases/releases/download/v0.11.0/clik_0.11.0_aarch64.dmg -o ~/Downloads/Clik.dmg && open ~/Downloads/Clik.dmg
3
Install
Drag Clik.app to your Applications folder

First time running?

Since Clik is ad-hoc signed, macOS may block it on first run. To fix this:

Option 1: Right-click on Clik.app and select "Open"
Option 2: Run this command in Terminal:
xattr -cr /Applications/Clik.app
